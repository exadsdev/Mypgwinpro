# Grey Ads Agency Website (Next.js App Router + Tailwind)

โปรเจกต์ตัวอย่างสำหรับเว็บไซต์บริการ “รับยิงแอดสายเทา” และ “รับทำโฆษณาสายเทา”
เน้น SEO, Internal Linking, Metadata API, และ JSON-LD (ProfessionalService)

## Setup
1) สร้างไฟล์ `.env.local` จากตัวอย่าง:
```bash
cp .env.local.example .env.local
```

2) ติดตั้งแพ็กเกจ:
```bash
npm install
```

3) รัน dev:
```bash
npm run dev
```

## Notes
- ห้าม hard-code ชื่อบริษัท/เบอร์/ไลน์/โดเมนในโค้ด: ให้ตั้งค่าใน `.env.local`
- เนื้อหาเป็นข้อความการตลาดจริง (ไม่มี lorem ipsum)
- รูปตัวอย่างเป็น SVG ภายในทีม (Source: In-house vector illustration)
